# Helper: `gqlClient` + `GqlClient`

## Why this exists

A typed GraphQL fetcher built around `@graphql-codegen/cli` with the **client preset** (`documentMode: 'string'`). It accepts the codegen-generated `TypedDocumentString<TResult, TVariables>` and gives you back fully typed result data. It also produces stable react-query `queryKey`s by extracting the operation name from the document, so cache invalidation is predictable.

The hook layer ([`use-gql-query.md`](use-gql-query.md)) sits on top of this.

## Where to put it

`src/core/gql-client/gql-client.ts`

## When to use it

- The data-fetching rules ([`docs/data-fetching.md`](../docs/data-fetching.md)) mandate `useGqlQuery` / `useGqlSuspenseQuery` / `useGqlMutation` for all GraphQL operations. Those hooks use this client. So: every GraphQL operation in the project goes through here, indirectly.
- Don't import `gqlClient.execute` directly in components — use the hooks. Direct use is reserved for non-React code (workers, scripts, server actions, etc.).

## Dependencies

```sh
yarn add graphql
yarn add -D @graphql-codegen/cli @graphql-codegen/client-preset
```

Plus the consumer's own:

- `@/core/gql/graphql` — generated by `codegen.ts` from a GraphQL schema. Run `yarn gql-codegen` after pointing it at your endpoint.
- `@/core/auth/auth.store` exporting `getAuthHeaders()` — returns `{ authorization: 'Bearer ...' }` (or empty). Wire to whatever auth lives in the consumer project.
- `@/core/config/config.service` exporting `config.gqlUrl` — the GraphQL endpoint. See [`unwrap`](unwrap.md) for the env-var pattern.

If your project already has different auth / config shapes, replace those imports at the top of the file with whatever you have. The class itself doesn't care — it just needs a `gqlUrl` and an optional `auth: () => Promise<Record<string, string>>`.

## Code

```ts
import { getOperationAST, parse } from 'graphql';

import { getAuthHeaders } from '@/core/auth/auth.store';
import { config } from '@/core/config/config.service';
import type { TypedDocumentString } from '@/core/gql/graphql';

type GqlClientParams = {
  gqlUrl: string;
  auth?: () => Promise<Record<string, string>>;
};

export type ExecuteOptions<TResult, TVariables> = {
  query: TypedDocumentString<TResult, TVariables>;
  variables?: TVariables extends Record<string, never> ? never : TVariables;
  skipAuth?: boolean;
};

export type GetQueryKeyOptions<TResult, TVariables> = {
  query: TypedDocumentString<TResult, TVariables>;
  variables?: TVariables extends Record<string, never> ? never : TVariables;
  includeVariables?: boolean;
};

class GqlClient {
  private readonly params: GqlClientParams;

  constructor(params: GqlClientParams) {
    this.params = params;
  }

  readonly execute = async <TResult, TVariables>(params: ExecuteOptions<TResult, TVariables>) => {
    const { query, variables, skipAuth } = params;

    let authHeaders: Record<string, string> = {};
    if (!skipAuth && this.params.auth) {
      authHeaders = await this.params.auth();
    }

    const headers = new Headers({
      'Content-Type': 'application/json',
      Accept: 'application/graphql-response+json',
      ...authHeaders,
    });

    const response = await fetch(this.params.gqlUrl, {
      method: 'POST',
      headers,
      body: JSON.stringify({ query, variables }),
    });

    if (!response.ok) {
      throw new Error('Network response was not ok');
    }

    const result = await response.json();
    if (result.errors) {
      throw result.errors;
    }

    return result.data as TResult;
  };

  readonly queryOptions = <TResult, TVariables>(options: ExecuteOptions<TResult, TVariables>) => {
    const { query, variables, ...rest } = options;
    const queryKey = this.getQueryKey({
      query,
      variables,
      includeVariables: true,
    });

    return {
      queryKey,
      queryFn: async () => this.execute(options),
      ...rest,
    };
  };

  readonly getQueryKey = <TResult, TVariables>(options: GetQueryKeyOptions<TResult, TVariables>) => {
    const { query, variables, includeVariables } = options;
    const name = getOperationAST(parse(query.toString()))?.name?.value;

    if (!name) {
      throw new Error('Query name is required');
    }

    const queryKey: (string | object)[] = [name];
    if (includeVariables && variables) {
      queryKey.push(variables);
    }

    return queryKey;
  };
}

export const gqlClient = new GqlClient({
  gqlUrl: config.gqlUrl,
  auth: () => getAuthHeaders(),
});
```
